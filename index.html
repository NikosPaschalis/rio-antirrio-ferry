Τελευταία ενημέρωση: Απρίλιος 2026

1. Ποιοι είμαστε

Η ιστοσελίδα rio-antirrio-ferry.netlify.app είναι ανεξάρτητη πληροφοριακή εφαρμογή για τα δρομολόγια του πορθμείου Ρίο–Αντίρριο. Δεν σχετιζόμαστε με καμία επίσημη εταιρεία πορθμείων ή λιμενική αρχή.

2. Δεδομένα που συλλέγουμε

Η ίδια η εφαρμογή δεν συλλέγει, δεν αποθηκεύει και δεν επεξεργάζεται προσωπικά δεδομένα. Δεν υπάρχουν φόρμες εγγραφής, λογαριασμοί ή προφίλ χρηστών.

3. Υπηρεσίες τρίτων

Η εφαρμογή χρησιμοποιεί τις παρακάτω υπηρεσίες τρίτων, οι οποίες ενδέχεται να επεξεργάζονται δεδομένα χρήστη (π.χ. διεύθυνση IP) σύμφωνα με τις δικές τους πολιτικές:

📊
Google Analytics
Χρησιμοποιείται για ανώνυμη στατιστική ανάλυση επισκεψιμότητας (αριθμός επισκεπτών, σελίδες, χώρες). Τα δεδομένα είναι ανωνυμοποιημένα. Πολιτική Google →
🌬️
Windfinder (iframe)
Ενσωματωμένος χάρτης καιρού/ανέμου. Ενδέχεται να τοποθετεί cookies τρίτων. Πολιτική Windfinder →
🚢
VesselFinder (Ζωντανή Παρακολούθηση)
Ενσωματωμένος χάρτης παρακολούθησης πλοίων. Ενδέχεται να τοποθετεί cookies τρίτων. Πολιτική VesselFinder →
☁️
Cloudflare Workers
Χρησιμοποιείται για λήψη δεδομένων μετεωρολογικού σταθμού. Το Cloudflare ενδέχεται να επεξεργάζεται διευθύνσεις IP για λόγους ασφαλείας και απόδοσης. Πολιτική Cloudflare →
🌤️
Open-Meteo API
Ανοικτό API καιρού — δεν συλλέγει προσωπικά δεδομένα. Όροι Open-Meteo →
4. Cookies

Η εφαρμογή δεν τοποθετεί ίδια cookies. Cookies ενδέχεται να προέρχονται από τις υπηρεσίες τρίτων που αναφέρονται παραπάνω (Google Analytics, Windfinder, VesselFinder). Μπορείτε να τα απενεργοποιήσετε μέσω των ρυθμίσεων του browser σας.

5. Διαφημίσεις

Στο μέλλον ενδέχεται να ενσωματωθούν διαφημίσεις μέσω Google AdSense. Σε αυτή την περίπτωση η πολιτική θα ενημερωθεί αντίστοιχα. Η Google ενδέχεται να χρησιμοποιεί cookies για εξατομικευμένες διαφημίσεις βάσει προηγούμενων επισκέψεων.

6. Αποποίηση ευθύνης

Τα δρομολόγια και τα δεδομένα καιρού παρέχονται αποκλειστικά για ενημερωτικούς σκοπούς. Δεν φέρουμε ευθύνη για τυχόν αλλαγές δρομολογίων λόγω καιρικών συνθηκών, τεχνικών προβλημάτων ή αποφάσεων λιμενικών αρχών. Επαληθεύετε πάντα απευθείας με τις αρχές.

7. Επικοινωνία

Για οποιοδήποτε ζήτημα σχετικά με την πολιτική απορρήτου, επικοινωνήστε μέσω της σελίδας rio-antirrio-ferry.netlify.app.
